# RockaVRDemo

用Unity3D 的UGUI构建一个VR场景，包含简单的一些交互和视角重置功能等，主要为前期调研成果。

### 交互功能
- 凝视交互横向填充
- 凝视交互圆圈填充
- 凝视交互Object 弹出


### 实际功能
- ViewGroup分页功能
- Pannel页面之间跳转
- MainCanvas和MainCamera视角重置功能
